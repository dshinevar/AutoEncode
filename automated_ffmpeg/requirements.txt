automated_ffmpeg Requirements

### Python3 ###
Requires Python3.
Written with Python 3.6.9

### pytz ###
For logging, automated_ffmpeg uses pytz to set the timezone and write the current time.

https://pypi.org/project/pytz/

pip3 install pytz

### plex_interactor ###
Uses the accompanying plex_interactor for updating plex.
Mainly just a really basic wrapper for a few functions from the
Python-PlexAPI.

Repo: https://github.com/pkkid/python-plexapi
Doc: https://python-plexapi.readthedocs.io/en/latest/

pip3 install plexapi
